# CRUD with Nodejs, Express and Mongodb

### Installation with docker-compose (Recommended)

```bash
git clone 
cd express-mongodb-crud
docker-compose up
```

### Installation (Manually)
#### Requirements

```bash
git clone 
cd express-mongodb-crud
npm install
npm run build
npm start
```

<a target="_blank" href="http://localhost:3000">http://localhost:3000</a>
